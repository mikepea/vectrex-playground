Mov_Draw_VLcs ($F3B5)

This routine moves to the first location specified in vector list, and then draws lines between the rest of coordinates in the list. The number of vectors to draw is specified as the first byte in the vector list, and the scale factor is the second byte in the vector list. The vector list has the following format:

count, scale, rel y, rel x, rel y, rel x, ...

 

ENTRY:

DP = $D0

X-reg points to the vector list

 

EXIT:

X-reg points to next byte after list

D-reg, X-reg trashed
