Moveto_ix ($F310)

This routine uses the current scale factor, and moves the pen to the (y,x) position pointed to by the X register. The X register is then incremented by 2.

 

ENTRY:

DP = $D0

X-reg points to the (y,x) coordinate pair

 

EXIT:

X-reg has been incremented by 2

D-reg trashed
