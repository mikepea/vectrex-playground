Rise_Run_Idx ($F5EF)

This routine gets the index pair for both the rise and run, using the passed-in angle value.

 

ENTRY:

DP = $C8

$C836 contains the angle value

 

EXIT:

$C837-$C838 contains the index pair for the run

$C839-$C83A contains the index pair for the rise

D-reg trashed
